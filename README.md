# python-task-manager
Mini console-based task manager un Python

A simple command-line task manager written in Python for learning purposes.

This project allows users to manage a list of tasks and mark them as completed.

## Features
- Add new tasks
- List all tasks
- Mark tasks as completed
- Simple and easy to understand codebase

## How to Run
1. Make sure you have Python 3 installed
2. Clone the repository
3. Run the main file:

## Example Usage
When you run the program, you can:
- Add tasks by providing a task description
- View all current tasks
- Mark a task as completed by selecting its index
